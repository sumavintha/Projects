Oscillation Model

This simulation visualizes the dynamic interactions between three biological systems: the Hypothalamus, Pituitary, and Gonads. These systems form a feedback loop that governs hormonal oscillations in the body. The parameter n represents the sensitivity of the feedback mechanism, controlling how sharply the system responds to changes in gonadal hormone levels. Adjust the n value using the slider below to observe its effect on the oscillation patterns.
